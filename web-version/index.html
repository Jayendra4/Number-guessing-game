Game Instructions

Thank you for playing Guess the Number game.

Here are the instructions for playing the game:

You have to guess the randomly generated number by computer in limited number of attempts. Each time you guess, the game will tell you if your guess is higher or lower than the target number.

  • Play game by pressing start button
  • There are two difficulty settings:
    • Easy: Number of attempts are 10
    • Hard: Number of attempts are 7
